API Reference

Complete API documentation for phpserialize-rs Python bindings.

loads()

Deserialize PHP serialized data to a Python object.

loads(data: bytes, *, errors: str = "replace", auto_unescape: bool = True) -> Any

Parameters

Name Type Default Description
data bytes - PHP serialized data
errors str "replace" Error handling for invalid UTF-8:
"strict" - Raise exception
"replace" - Replace with U+FFFD
"bytes" - Return bytes instead of str
auto_unescape bool True Automatically detect and unescape DB-exported strings

Returns

dict | list | str | int | float | bool | None | bytes

The deserialized Python object. Arrays become list (sequential) or dict (associative).

Raises

PhpDeserializeError - If the data cannot be parsed

Example

from php_deserialize import loads

# Basic usage
result = loads(b'a:2:{s:4:"name";s:5:"Alice";s:3:"age";i:30;}')
# {'name': 'Alice', 'age': 30}

# Korean text
result = loads(b's:6:"\xed\x95\x9c\xea\xb8\x80";')
# '한글'

# Invalid UTF-8 handling
result = loads(b's:4:"\xff\xfe\xfd\xfc";', errors="bytes")
# b'\xff\xfe\xfd\xfc'

loads_json()

Deserialize PHP serialized data directly to a JSON string. Optimized for Databricks/Spark workloads.

loads_json(data: bytes, *, auto_unescape: bool = True) -> str

Parameters

Name Type Default Description
data bytes - PHP serialized data
auto_unescape bool True Automatically detect and unescape DB-exported strings

Returns

str - JSON string representation

Example

from php_deserialize import loads_json

json_str = loads_json(b'a:2:{s:4:"name";s:5:"Alice";s:3:"age";i:30;}')
# '{"name":"Alice","age":30}'

is_serialized()

Quick check if data looks like PHP serialized format (doesn't fully validate).

is_serialized(data: bytes) -> bool

Example

from php_deserialize import is_serialized

is_serialized(b'a:1:{i:0;s:3:"foo";}')  # True
is_serialized(b'not serialized')         # False
is_serialized(b'')                       # False

preprocess()

Preprocess data to unescape DB-exported strings manually.

preprocess(data: bytes) -> bytes

Example

from php_deserialize import preprocess

escaped = b'"a:1:{s:3:""key"";s:5:""value"";}"'
unescaped = preprocess(escaped)
# b'a:1:{s:3:"key";s:5:"value";}'

PhpDeserializeError

Exception raised when PHP deserialization fails.

Example

from php_deserialize import loads, PhpDeserializeError

try:
    loads(b"invalid data")
except PhpDeserializeError as e:
    print(f"Error: {e}")

php_deserialize_udf()

Create a PySpark UDF for deserializing PHP serialized data.

php_deserialize_udf(output_format: str = "json", errors: str = "replace", auto_unescape: bool = True) -> Callable

Parameters

Name Type Default Description
output_format str "json" "json" returns JSON string, "python" returns Python object
errors str "replace" Error handling mode

Example

from php_deserialize.spark import php_deserialize_udf

deserialize = php_deserialize_udf("json")
df = df.withColumn("parsed", deserialize("php_data"))

php_deserialize_pandas_udf()

Create a batch-optimized Pandas UDF (recommended for large datasets).

php_deserialize_pandas_udf(errors: str = "replace", auto_unescape: bool = True) -> Callable

Example

from php_deserialize.spark import php_deserialize_pandas_udf

pandas_udf = php_deserialize_pandas_udf()
df = df.withColumn("parsed", pandas_udf("php_data"))

Rust: from_bytes()

Parse PHP serialized data from bytes.

pub fn from_bytes(data: &[u8]) -> Result<PhpValue<'_>>

Example

use php_deserialize_core::from_bytes;

let data = b"s:5:\"hello\";";
let value = from_bytes(data)?;
